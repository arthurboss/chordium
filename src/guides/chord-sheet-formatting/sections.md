# Sections

To separate different sections of the song (Intro, Verse, Chorus, etc.), use a blank line followed by the section name in square brackets.

&nbsp;

```
[Intro]
[C]      [G]      [Am]
Let it be, let it be, let it be

[Verse 1]
[C]      [G]      [Am]
When I find myself in times of trouble
